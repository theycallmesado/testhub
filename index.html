Git commands: git clone (copies the repository to your local files) git status (checks the status) git add (adds a file to the repository) git add -A (adds all files to the repository) git commit -m "message" (commits the added files to the repository) git push (pushes the commited files to repository) git pull (pulls the online files to your local repository) [ESC] + :wq (in case you commit without a message)
